# Backend
## Fill appsettings with needed configuration, apply migrations to db
```json
{
  "Logging": {
    "LogLevel": {
      "Default": "Information",
      "Microsoft.AspNetCore": "Warning"
    }
  },
  "ConnectionStrings": {
    "DefaultConnection": ""
  },
  "Origin": ""
}

```
## Run API

# Frontend

ng serve
